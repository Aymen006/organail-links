A premium "link-in-bio" landing page for **Organail London** — salon-quality nails, without the salon.

Built as a single self-contained static page (no build step) for a one-place hub linking to the shop and every social channel, designed to be scanned from a QR code on business cards.

## Design
- Soft-luxury beauty aesthetic: porcelain `#F7F4EF` + warm nude `#C9A892`, ink `#1A1A1A`
- Type: **Fraunces** (serif display) + **Inter** (sans)
- Motion: staggered entrance reveals, animated aurora background + grain, magnetic hover, shimmer sweep on the Shop CTA, tap ripple — all respecting `prefers-reduced-motion`
- Fully responsive, mobile-first, fast (static, no framework)

## Tech
Plain `index.html` + `styles.css` + `script.js`. Deployed on Vercel.
